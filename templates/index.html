FAQ

Common questions

Is the free subdomain really permanent? ∨
Yes. yourname.devdrop.sh is yours for as long as your account is active, including on the free tier. We don't reclaim subdomains unless an account is deleted.
How does GitHub Pages deployment work? ∨
DevDrop generates an index.html listing all your uploaded files with correct public links. Download it, push to a GitHub repo, and enable Pages under Settings → Pages. Done.
What file types are supported? ∨
Any file type. No extension restrictions. HTML, CSS, JS, ZIP, PDF, PNG, JPG, SVG, MP4, WebM, JSON, WASM — all work with correct MIME types served automatically.
Can I use DevDrop in CI/CD? ∨
Yes. Pro and Team plans include REST API access. Upload files via POST /api/v1/upload with an API key. A GitHub Action is also available in the marketplace for zero-config CI integration.
How is bandwidth counted? ∨
Storage quota refers only to stored file size. Bandwidth (downloads served) is not counted toward your limit on any plan — you can serve unlimited downloads from hosted files.
Can I password-protect specific files? ∨
Yes, on Pro and Team plans. Set a password per-file — visitors will be prompted before downloading. You can also set expiry dates so files automatically become inaccessible after a chosen date.
Does DevDrop support direct video download? ∨
The video manager stores URLs you provide. For direct .mp4 links hosted on DevDrop, a download button is shown. For platform URLs (YouTube, Vimeo), DevDrop opens them in a new tab — platform Terms of Service apply.
Can I cancel anytime? ∨
Absolutely. Cancel from your dashboard at any time. You keep Pro/Team access until end of billing period. Annual plans are refundable within 30 days of purchase.
